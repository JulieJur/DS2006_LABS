# DS2006_LABS

### Introduction to Data Science – Laboratory Collection

This repository contains all laboratory assignments for the course **DS2006 – Introduction to Data Science**.
Each lab is organized in a separate folder with clear structure and naming according to the course instructions.

---

## Lab 1 – Introduction to Python

**Folder:** `Lab1_Intro_to_Python`

**Purpose:**

- Learn the basics of Python syntax and logic.
- Use variables, arithmetic operations, conditionals, loops, and user input.
- Practice running Python code both in `.py` files and Jupyter notebooks.

**Files:**

- `lab1.py`
- `lab1.ipynb`

---

## Lab 2 – Dice Problem

**Folder:** `Lab2_Dice_Proble`

**Purpose:**

- Use randomization and simulation in Python.
- Work with the `random` library to simulate dice rolls.
- Extend the program to roll different dice (e.g., D6, D20) and show results.

**Files:**

- `dice_Simulation.py`

---

## Lab 3 – Battle of Dices

**Folder:** `Lab3_Battle_of_Dices`

**Purpose:**

- Create an interactive dice game between two players.
- Apply conditionals, loops, and modularization.
- Refactor code across multiple tasks (1–14) as described in the lab instructions.
- Implement separate files for each task to demonstrate incremental development.

**Files:**

- `BattleOfDices Task-1.py` → `battle-of-dices-better-task14.py`
- `BattleOfDices Reflection Task-9.txt`
- `MyDice.py` (utility functions for dice)
